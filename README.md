# 🗄️ 디비디비-딥 (DBDB-Deep)

> **"데이터베이스의 깊은 곳까지 내려가는 딥다이브 스터디"**  
> 우아한테크코스 8기 크루들이 모여 각자의 DB 역량을 강화하기 위해 운영되는 스터디입니다.

---

## 🎯 스터디 목표

- **DB 실력 키우기**: DB의 내부 동작 원리와 최적화 방법을 깊이 있게 공부합니다.
- **꾸준히 하기**: 2주 단위로 개인별 학습 목표를 세우고 꾸준히 기록합니다.
- **함께 성장하기**: 서로 배운 내용을 공유하고 피드백하며 지식을 나눕니다.

---

## ⚙️ 운영 방식

1. 각자 정한 도서나 강의를 2주 동안 공부하고 기록합니다.
2. 매주 정해진 시간에 모여서 그룹별로 발표 세션을 갖습니다.
3. 발표 시간은 20~30분 정도로, 자유로운 형식으로 진행합니다.
4. 발표 후에는 자유롭게 질문하고 의견을 주고받으며 피드백을 나눕니다.
5. 모든 멤버가 본인의 목표를 달성할 때까지 스터디를 지속합니다.
